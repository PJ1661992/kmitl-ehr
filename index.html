⚙️ ตั้งค่าเชื่อมต่อ Google Sheets

📋 วิธีการตั้งค่า:

  1. เปิด Google Cloud Console → สร้าง API Key
  2. เปิดใช้งาน Google Sheets API
  3. ตั้งค่า API Key ให้จำกัดเฉพาะ Sheets API
  4. ตรวจสอบว่า Google Sheets ตั้งเป็น Public (anyone with link)
  5. ใส่ Spreadsheet ID และ API Key ด้านล่าง แล้วกด บันทึก

พบได้ใน URL ของ Google Sheets: .../d/[ID ที่นี่]/edit

API Key เก็บใน localStorage ของเบราว์เซอร์เท่านั้น ไม่มีการส่งออกไปที่อื่น
